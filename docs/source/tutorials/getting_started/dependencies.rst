Dependencies
------------

In general, this project requires specific versions for the following dependencies:

- python version >3.8

- Sphinx version >1.8.5
 
- Python dependencies

Install the dependencies:

.. code-block::

  python3 -m pip install -r <path_to_requirements_file>
